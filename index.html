Markdown 基本语法

1、标题

1.1 使用 = 和 - 标记一级和二级标题

使用:

This is an H1     
=============     
This is an H2
-------------

效果:

This is an H1

This is an H2

1.2 使用 #、##、###、####、#####、###### 标记 一二三四五六级标题

使用:

#一级标题
##二级标题
###三级标题
####四级标题
#####五级标题
######六级标题

效果:

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

2、正文

2.1 设置斜体、粗体、删除线

使用:

*这是文字*      斜体
_这是文字_      斜体
**这是文字**    粗体
***这是文字***  粗体+斜体
~~这是文字~~    删除线

效果:

这是文字 斜体
这是文字 斜体
这是文字 粗体
这是文字 粗体+斜体
这是文字 删除线
